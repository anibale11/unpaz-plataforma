Instalar en Docker Compose


En Caso de que tenga Linux en su Computadora tambien necesita instalar docker-compose

// Descargar Desde El Repositorio

'sudo curl -L "https://github.com/docker/compose/releases/download/1.29.2/docker-compose-$(uname -s)-$(uname -m)" -o /usr/local/bin/docker-compose'

// Dar Permisos de Ejecucion

'sudo chmod +x /usr/local/bin/docker-compose'

// Crear un enlace simbólico (similar a Acceso Directo en Windows)

'sudo ln -s /usr/local/bin/docker-compose /usr/bin/docker-compose'

// Si tienes todo instalado puedes testear la version de docker compose

'docker-compose --version'
